Simple minesweeper game, still working on it.

The front end part of it was made in bootstrap5. For the javascript, I used 2 API :

  • https://random-words-api.herokuapp.com/ (to retrieve random word)
  • https://dictionaryapi.dev/ (to retrieve the definition of a word )
